Room Swapping Guidelines

Welcome!

Here's a quick overview of how our room swapping system works:

1. Listing Room Availability

  • Registered students can list their rooms as available for swapping.

2. Requesting a Swap

  • Students can view available rooms and send swap requests to other students.

3. Accepting or Rejecting Requests

  • Students with available rooms can accept or reject swap requests.
  • If a request is accepted, the room swap takes place. If rejected, the room remains available.

4. Automatic Rejection

  • If a swap request is accepted, pending requests for the same room are automatically rejected.

5. Finalizing Swaps

  • Once a swap is accepted, both parties receive notifications of the successful swap.

For any questions or assistance, please contact our support team.
